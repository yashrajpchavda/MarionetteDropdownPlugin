DropdownBodyView Class

Extends Marionette.CompositeView
Defined in: views/ddBodyCompView.js:1
Module: Dropdown

The composite body view for the dropdown items, used to render the items in the dropdown list view. Responsible for various things like, refreshing the items as and when user types. Uses DropdownBodyItemView as a childView and DropdownBodyItemsCollection as a collection for the view.

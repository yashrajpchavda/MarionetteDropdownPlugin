DropdownSelectedTagsView Class

Extends Marionette.CompositeView
Defined in: views/selectedTagsCompView.js:1
Module: Dropdown

A view that renders all the selected items tags and the input box to search the items. It has different behaviour based on single select or multi select. Renders itself accordingly. Uses DropdownSelectedTagItemsCollection as a collection and DropdownSelectedTagItemView as an childView.
